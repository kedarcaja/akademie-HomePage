# 15 – Roadmapa implementace

## Fáze 1 – Launch (základní funnel)

**Cíl:** Začít sbírat e-maily a prodat první kurz.

- [ ] Homepage (Akademie) s hero a lead magnetem
- [ ] Funnel „7 dní u štětce" + e-mailová sekvence (5 e-mailů)
- [ ] Stránka Začni tady se 4 branami
- [ ] Kurz „První plamen" (základní moduly)
- [ ] Komunita Dračí kruh (základní nastavení)
- [ ] Stránka O nás
- [ ] Tagy: interest-zacatecnik, access-prvni-plamen, level-vejce-draka

**Délka:** 4–6 týdnů

---

## Fáze 2 – Rozšíření

**Cíl:** Pokrýt všechny hlavní cílové skupiny.

- [ ] Stránka Děti a rodiče + funnel „Tvoření mimo obrazovky"
- [ ] Stránka Školy + poptávkový formulář
- [ ] Stránka Stezky s přehledem levelů
- [ ] Kurzy: Vejce draka, Mládě draka
- [ ] E-mailové sekvence: rodiče, školy, členství
- [ ] Achievementy (poloruční workflow)
- [ ] Obchod se členstvím

**Délka:** 6–10 týdnů

---

## Fáze 3 – Komunita a obsah

**Cíl:** Aktivní komunita, pravidelný obsah, škálování.

- [ ] Dračí rádio (podcasty jako lekce)
- [ ] Meditační zahrada (audio zklidnění)
- [ ] Měsíční témata jako kampaně
- [ ] Galerie pokroku (komunita + externní)
- [ ] Pokročilé kurzy: Učeň ohně, Strážce pokladu
- [ ] Workshopy škol (kalendář, rezervace)
- [ ] Malířský deník (Notion šablona nebo PDF)

**Délka:** průběžně

---

## Fáze 4+ – Vlastní vývoj

Pouze pokud komunita roste a ruční procesy nestačí:

- [ ] Automatický achievement engine
- [ ] Profil studenta s historií pokroku
- [ ] Žebříčky a XP systém
- [ ] AI mentor (integrace Claude API)

---

## Prioritizace obsahu

**Nejdůležitější pro launch:**
1. Homepage hero + lead magnet
2. E-mailová sekvence 7 dní
3. Kurz První plamen (min. 3 moduly)
4. Komunita Dračí kruh

**Může počkat:**
- Bestiář technik (obsah pro stávající studenty)
- Certifikáty a pokročilé achievementy
- Měsíční témata jako plánované kampaně
