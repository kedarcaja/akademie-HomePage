# 02 – Veřejný web vs. členská sekce

## Klíčové pravidlo

> Veřejný web = přesvědčit a orientovat. Členská sekce = vzdělávat a zapojit.

**Na veřejném webu NESMÍ být:**
- Kompletní seznam všech kurzů a lekcí
- Achievement systém a gamifikace
- Podrobnosti o každé stezce
- Technické detaily pro specifické skupiny (deskovkáři, display malíři)
